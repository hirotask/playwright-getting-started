# playwright-getting-started

このリポジトリはE2Eテストツールである[Playwright](https://playwright.dev/) を実際に触って楽しむために作成されました。

## インストール

### 前提条件

以下の環境が整っている必要があります。

- Node.js >= v20.12.0
- npm >= 10.5.0

### 環境構築

1. `npx playwright install-deps` を実行してOSに依存するライブラリを手動でインストールします
2. `npm run test:e2e` を実行してテストを実行します。
